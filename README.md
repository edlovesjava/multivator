# multivator
Multiple agent innovation based on an RFP and guided process of teams of agents innovating independently and competing for the contract

## Implementation

- See [agent-team-scaffold](./agent-team-scaffold)
- Leverages [claude code agent teams](https://code.claude.com/docs/en/agent-teams)
