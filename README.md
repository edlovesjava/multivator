# multivator

Multiple agent innovation based on an RFP and guided process of teams of agents innovating independently and competing for the contract.

## Repository Structure

- **[multivator-skills/](multivator-skills/)** — The framework: composable bidder profiles, RFP skills, team constitution, and agent rules
- **[demo/](demo/)** — End-to-end proof-of-concept using the MeshSwarm RFP (see [demo/README.md](demo/README.md) for the full walkthrough: RFP input, execution process, implementor artifacts, judge verdict, and synthesized recommendation)
