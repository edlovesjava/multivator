# multivator
Multiple agent innovation based on an RFP and guided process of teams of agents innovating independently and competing for the contract

## demo/

Contains the output from our proof-of-concept run using the `multivator-skills`. Two independent teams (A and B) responded to the MeshSwarm RFP, each producing approach documents, cost analyses, interface designs, privacy analyses, power analyses, test strategies, and self-reviews.

- `mesh-swarm.md` — the RFP spec used as input
- `claude-agent-teams-paper.md` — writeup on the agent-team process
- `APPROACH.md`, `COST-ANALYSIS.md`, `INTERFACES.md`, `POWER-ANALYSIS.md`, `PRIVACY-ANALYSIS.md`, `TEST-STRATEGY.md`, `SELF-REVIEW.md` — Team A output
- `B-APPROACH.md`, `B-COST-ANALYSIS.md`, `B-INTERFACES.md`, `B-POWER-ANALYSIS.md`, `B-PRIVACY-ANALYSIS.md`, `B-TEST-STRATEGY.md`, `B-SELF-REVIEW.md` — Team B output
