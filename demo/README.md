# Demo: MeshSwarm RFP — Agent Team Proof of Concept

This directory contains the full output from a proof-of-concept run of the `multivator-skills`. Two independent Claude Code agent teams (A and B) each responded to a realistic RFP for a self-organizing AIoT mesh network (MeshSwarm), then a judge agent evaluated both proposals.
